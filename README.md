# 🧪 STM LABS Test Task

## 📌 About
This project was developed as a test assignment for STM LABS.  
It demonstrates frontend development skills using React, TypeScript, SCSS, and Vite, with a focus on clean architecture, responsive UI, and modern development practices.

## 🌐 Live Demo
👉 [Open Project](https://wirtrax.github.io/STMLabs/)

## ✨ Features
- ⚛️ Built with React
- 🧩 Component-based architecture
- 🎨 Styling with SCSS
- ⚡ Fast development and optimized build with Vite
- 📱 Responsive user interface
- 🛠 Clean and maintainable code structure
- 🚀 Modern frontend development approach
